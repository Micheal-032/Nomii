🧠 Nomii – AI-Powered Smart Collaboration Platform

Nomii is an intelligent platform that connects suppliers, farmers, artisans, and delivery partners into a unified ecosystem.
Using AI automation, smart logistics, and an advanced AI chatbot, Nomii simplifies supply chain collaboration, improves productivity, and 
enables data-driven decision making.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

📘 Project Overview

Nomii enables multiple stakeholders to collaborate efficiently on one platform. It integrates AI recommendations, automation systems, analytics dashboards,
and conversational AI** to optimize sourcing, production, and delivery workflows.
The platform focuses on **smart coordination, predictive insights, and automated logistics** to reduce inefficiencies and improve operational transparency.

⚙️ Setup & Installation

Follow these steps to run Nomii locally.

🧩 Prerequisites

• Python 3.8+
• Node.js (for frontend)
• Git
• Modern web browser

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

🚀 Clone the Repository

1) Open your terminal and run:

```
git clone https://github.com/your-username/nomii.git
cd nomii
```

---

📦 Backend Setup

2) Create a virtual environment:

```
python -m venv venv
```

3) Activate it:

Windows

```
venv\Scripts\activate
```

Mac/Linux

```
source venv/bin/activate
```

4) Install dependencies:

```
pip install -r requirements.txt
```

---

🔑 Environment Variables (API Keys)

5) Create a `.env` file in the root directory and add your API keys.

```
OPENAI_API_KEY=your_openai_api_key
HUGGINGFACE_API_KEY=your_huggingface_key
MAPBOX_API_KEY=your_mapbox_key
OPENWEATHER_API_KEY=your_weather_api_key
SENDGRID_API_KEY=your_sendgrid_key
TWILIO_API_KEY=your_twilio_key
```

---

6) ▶️ Run the Backend Server

```
python app.py
```

Server will start at:

```
http://localhost:5000
```

---

💻 Frontend Setup

7) Navigate to frontend folder:

```
cd frontend
npm install
npm start
```

Frontend will run at:

```
http://localhost:3000
```
